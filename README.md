## 12. [Deterministic Fully-Static Whole-Binary Translation Without Heuristics](https://news.ycombinator.com/item?id=48117810) (↑256, 💬61)

It considers all possible byte interpretations to produce deterministic translations, pruning only those leading to abnormal termination, achieving performance on par with or better than QEMU. **Comments:** Commenters discuss the trade-off of a 50x code size increase for deterministic translation, noting its value for regulated industries requiring certified binaries.
